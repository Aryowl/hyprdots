# Dependencies
- hyprland
- hyprpaper
- hyprlock
- hypridle
- hyprpicker (optional)
- vim
- cava
- waybar
- rofi
- nvim (optional)

# Installation
Clone the Repo and drop the configuration files in ```~/.config``` directory
